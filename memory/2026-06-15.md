### 05:18 - 05:23 - 重置工作区
操作步骤：
1. `openclaw gateway stop` — 停止 gateway
2. `Remove-Item *.md -Force` — 删除 workspace 中所有自定义 md 文件
3. `Remove-Item memory -Recurse -Force` — 删除 memory 目录
4. `openclaw setup` — 重新生成默认文件（遇到 WorkspaceVanishedError）
5. 删除 `workspace-attestations` 中的认证文件
6. 再次 `openclaw setup` — 成功
