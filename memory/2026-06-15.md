- 建议：指定具体模型（如 `openrouter/anthropic/claude-sonnet-4`）

### 05:12 - 用户决定重置
- 用户认为之前把工作区搞乱了（克隆了自定义仓库覆盖了默认模板）
- 自定义 SOUL.md 包含大量安全协议，和默认版差异很大

### 05:18 - 05:23 - 重置工作区
操作步骤：
1. `openclaw gateway stop` — 停止 gateway
2. `Remove-Item *.md -Force` — 删除 workspace 中所有自定义 md 文件
3. `Remove-Item memory -Recurse -Force` — 删除 memory 目录
4. `openclaw setup` — 重新生成默认文件（遇到 WorkspaceVanishedError）
5. 删除 `workspace-attestations` 中的认证文件
6. 再次 `openclaw setup` — 成功

### 05:23 - 05:26 - 清空会话记录
- `Remove-Item sessions\* -Recurse -Force` — 清空所有历史会话
- Dashboard webchat 记录需要在浏览器 Local Storage 中清除

### 05:27 - 05:31 - Gateway 服务管理

- 后发现 dashboard 需要服务，重新安装回来
